This repository contains a dual-language implementation of the Quick Sort algorithm, written in both Python and Java, to demonstrate core algorithmic concepts and efficiency paradigms. Developed as part of a computer science foundation curriculum, the project highlights two distinct approaches to implementing the same divide-and-conquer strategy. The Python script leverages highly expressive list comprehensions to cleanly partition elements into three separate sub-arrays—less than, equal to, and greater than the chosen middle pivot—before recursively merging them. Conversely, the Java implementation demonstrates a more memory-efficient, in-place sorting mechanism using standard pointers, low/high indices, and a dedicated partition method that swaps primitive integers within a single array using a rightmost pivot. By showcasing both functional-style array filtering and classic mutable index manipulation, this project serves as an excellent comparative study of software architecture, memory management, and recursion handling across different programming languages.
